# Single-Axis-Solar-Tracking-System-with-hardware-only
developed an analog single-axis solar tracking system aimed at maximizing the power output of a photovoltaic panel.

 Built from the ground up, the project covered:
 • Circuit design & experimental validation
 • PCB development
 • Mechanical structure modeling

Using light-dependent sensors and analog control circuitry, the system continuously adjusts the panel’s orientation to follow the sun throughout the day.

Performance evaluation showed a power increase of approximately 56% when compared to a static solar panel
